## Wordmark Font

Use a rounded, high-weight sans stack for the Placebook name:

`Avenir Next Rounded, Avenir Next, Nunito Sans, Inter, Arial Rounded MT Bold, sans-serif`

The app component uses native equivalents: Avenir Next on iOS, `sans-serif-medium` on Android, and the full stack on web.
